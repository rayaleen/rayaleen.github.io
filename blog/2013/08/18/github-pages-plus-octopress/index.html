Github Pages + Octopress

发现现在很多git用户都开始使用github pages + octopress来建立自己的blog。Octopress宣扬的是blogging framework for hacker,如果懂点shell commands和基本的git用法就可以使用。所以自己也打算尝试一下并且开启自己的blog。

Octopress Setup

Requirement

  • Install Git,强烈推荐使用Homebrew来安装git
  • Install Ruby 1.9.3 using RVM

    $ \curl -L https://get.rvm.io | bash -s stable —ruby =1.9.3
